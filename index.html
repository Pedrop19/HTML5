HTML 5.

(HyperText Markup Language, versión 5)

Introducción

HTML 5 es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 específica dos variantes de sintaxis para HTML: una «clásica», HTML (text/html), conocida como HTML5, y una variante XHTML conocida como sintaxis XHTML 5 que deberá servirse con sintaxis XML (application/xhtml+xml).1​2​ Esta es la primera vez que HTML y XHTML se han desarrollado en paralelo. La versión definitiva de la quinta revisión del estándar se publicó en octubre de 2014.3

Diferencias entre HTML5 y HTML4/XHTML

  • HTML
  • Es nativo, y por tanto independiente
  • Es más semantico
  • El Código es más simple
  • Mayor Compatibilidad con dispositivos móviles
  • Incluye la etiqueta Canvas
  • HTML5
  • Nuevos elementos
  • Descubrir con mayor precisión el contenido
  • Nuevas formas de comunicarse con el servidor
  • Posibilidad de almacenamiento local
  • Excelente soporte para contenido multimedia

Principales Novedades

Incorpora etiquetas Canvas 2D y 3D, audio, vídeo
